# electrical_equipment
* задаче №1 произведён расчет электрических нагрузок и токов по заданной мощности. На основе полученного тока выбрано сечение питающих кабелей по допустимому току и проверено условие надежности. Далее произведён выбор трансформаторов с учетом категории электроснабжения и необходимого резерва мощности. Также выполнен подбор предохранителей для защиты сети от токов короткого замыкания на стороне высокого и низкого напряжения.

* В задаче №2 определена суммарная установленная мощность оборудования цеха и расчетная мощность с учетом коэффициента спроса. На основании этого выбрано количество силовых пунктов и их размещение. Далее рассчитаны токи нагрузки, по которым произведён выбор сечений кабелей для питания силовых пунктов и отдельных станков. Также выбраны аппараты защиты (автоматы) для ввода и распределительных линий.

* Все расчёты выполнены по стандартным формулам электротехники с учетом коэффициента мощности, КПД и допустимых токовых нагрузок проводников
